Adds a key and gutenberg block schema within the WordPress REST API

== Description ==

This plugin adds a block key within the [WordPress REST API](https://developer.wordpress.org/rest-api/) with valid Gutenberg JSON schema.

== Installation ==

1. Activate the plugin through the 'Plugins' screen in WordPress
